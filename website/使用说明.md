## 🚀 快速开始

### 方式一：仅查看网页（最简单）

1. **打开文件**
   ```
   双击 website/index.html
   ```
   或者
   ```bash
   cd G:\yige\website
   start index.html
   ```

2. **完成！**
   - 网页会在浏览器中自动打开
   - 可以浏览所有内容
   - 图片会正常显示（独立的5张干净图片）

### 方式二：运行完整网站（含后端功能）

1. **安装Python依赖**
   ```bash
   cd G:\yige\website
   pip install -r requirements.txt
   ```

2. **启动服务器**

   **Windows用户**（推荐）：
   ```bash
   双击 start_server.bat
   ```

   **或手动启动**：
   ```bash
   cd backend
   python app.py
   ```

3. **打开浏览器**
   ```
   访问: http://localhost:5000
   ```

4. **看到以下信息说明成功**：
   ```
   ======================================================================
   ContrailTracker Backend API Server
   ======================================================================
   Server starting on http://localhost:5000
   ======================================================================
   ```

## 🔧 后端服务

### API功能（需要启动后端）

#### 启动后端后可用的功能：

1. **文件上传分析**
   - 在Demo区域上传卫星和航班数据
   - 点击"Run Analysis"进行分析
   - 查看实时结果

2. **联系表单**
   - 填写联系表单
   - 提交后信息会保存到服务器

3. **API调用**
   - 可以通过代码调用API
   - 获取分析结果

### API端点列表

```
GET  /api/health              # 健康检查
POST /api/analyze/contrail    # 尾迹云分析
POST /api/analyze/emissions   # 排放计算
POST /api/analyze/carbon-markets  # 碳市场对比
POST /api/upload              # 文件上传
POST /api/contact             # 联系表单
GET  /api/stats               # 平台统计
```

### 测试API

```bash
# 健康检查
curl http://localhost:5000/api/health

# 获取统计数据
curl http://localhost:5000/api/stats
```

## 🎨 自定义修改

### 1. 修改颜色主题

编辑 `website/css/style.css` 文件：

```css
:root {
    /* 修改这些颜色 */
    --primary-blue: #0066cc;      /* 主蓝色 */
    --secondary-green: #00cc66;   /* 主绿色 */
    --dark-bg: #0a1628;           /* 背景色 */
}
```

### 2. 修改团队成员信息

编辑 `website/index.html`，找到 Team Section (大约第530行)：

```html
<div class="team-card">
    <div class="team-info">
        <h3>你的名字</h3>
        <span class="team-role">你的角色</span>
        <p>你的个人简介...</p>
        <div class="team-skills">
            <span>技能1</span>
            <span>技能2</span>
            <span>技能3</span>
        </div>
    </div>
</div>
```

### 3. 修改联系信息

编辑 `website/index.html`，找到 Contact Section (大约第640行)：

```html
<div class="contact-method">
    <i class="fas fa-envelope"></i>
    <div>
        <strong>Email</strong>
        <span>你的邮箱@example.com</span>
    </div>
</div>
```

### 4. 添加/替换图片

1. 将新图片放到 `website/images/` 目录
2. 编辑 `index.html`，找到对应的 `<img>` 标签
3. 修改 `src` 属性：
   ```html
   <img src="images/你的图片名.png" alt="描述">
   ```

### 5. 修改统计数字

编辑 `website/index.html`，找到 Hero Section 的统计区域：

```html
<div class="stat-number" data-target="2197">0</div>
<div class="stat-label">Tonnes CO₂ Tracked</div>
```

修改 `data-target` 的值为你想要的数字。

## ❓ 常见问题

### Q1: 图片显示不出来？

**答**: 检查以下几点：
1. 确保图片在 `website/images/` 目录下
2. 检查文件名是否正确（区分大小写）
3. 如果直接打开HTML，确保使用相对路径 `images/xxx.png`

### Q2: 后端启动失败？

**答**:
```bash
# 1. 检查Python版本（需要3.8+）
python --version

# 2. 重新安装依赖
pip install -r requirements.txt --force-reinstall

# 3. 检查端口是否被占用
netstat -ano | findstr :5000
```

### Q3: 网页显示乱码？

**答**:
- 确保文件编码为UTF-8
- 使用记事本打开时选择"UTF-8"编码保存

### Q4: 动画不流畅？

**答**:
- 使用Chrome或Edge浏览器（推荐）
- 避免使用IE浏览器
- 确保硬件加速已开启

### Q5: 如何在其他电脑上使用？

**答**:
1. **仅前端**: 复制整个 `website` 文件夹即可
2. **含后端**:
   - 复制 `website` 文件夹
   - 在新电脑上安装依赖
   - 启动服务器

### Q6: 可以部署到网上吗？

**答**: 可以！

**纯前端部署**（推荐初学者）：
- GitHub Pages（免费）
- Netlify（免费）
- Vercel（免费）

**完整部署**（前端+后端）：
- Heroku（免费套餐）
- Railway（免费套餐）
- 自己的服务器

### Q7: 如何分享给他人？

**答**:
1. **本地查看**: 直接发送 `website` 文件夹
2. **在线分享**: 部署到GitHub Pages后发送链接
3. **演示**: 启动服务器，同一WiFi下访问 `http://你的IP:5000`

### Q8: 表单提交后数据去哪了？

**答**:
- 如果启动了后端，数据会保存到 `website/backend/uploads/contact_submissions.log`
- 如果没启动后端，表单会显示错误（这是正常的）

### Q9: 如何修改网站标题？

**答**:
编辑 `index.html` 的 `<title>` 标签：
```html
<title>你的网站名称</title>
```

### Q10: 移动端能用吗？

**答**:
- ✅ 完全支持！网站已做响应式设计
- 在手机浏览器中打开即可
- 导航菜单会自动变成汉堡菜单
